# Pricing Pattern Shift in Real Estate Post-Pandemic

## Objective
Analyze real estate pricing trends before and after the COVID-19 pandemic and identify changes across property types.

## Tools Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Jupyter Notebook

## Analysis Highlights
- Cleaned and preprocessed 20,000+ real estate records
- Categorized data into pre- and post-pandemic periods
- Compared average sale prices across property types

## Key Insights
- Post-pandemic prices are higher or comparable across most property types
- Apartments and Industrial properties show significant growth
- Real estate market demonstrates strong post-pandemic resilience
